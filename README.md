# SecurusAudire
### A Security Audit Tool
---



SecurusAudire auditing tool for linux like systems and windows
- Branches - 
  - **master** branch contains the combined tool for both windows and linux like systems.
  - **linux_dev** contains the audit tool only for linux like systems.
  - **windows_dev** contains the audit tool only for windows systems.

Software requirments - python3 

To run SecurusAudire - 
1. pip install -r requirements.txt
2. sudo python3 SecurusAudire[GUI/CLI].py

Default storage location for audit results -
- **windows - Root_Drive/SecurusAudire_Reports**
- **linux - /var/log/SecurusAudire_Reports**
